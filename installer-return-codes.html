SkillHour Setup — EXE return codes

SkillHour-Setup is an NSIS installer for Windows (x64). Silent install: /S

Standard process exit codes:

Code Meaning
0 Success — installation completed normally
1 Installation cancelled by the user
2 Installation aborted by the installer script

Package URL example: https://www.skillhour.net/downloads/1.0.0/SkillHour-Setup-1.0.0.exe
